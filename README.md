# LLM-Fine-Tuning
This repository is used for documenting my exploration of LLM and AI models using Hugging face. 
1. A rough LLM was implemented and tested on. I explored generation, fine-tuning and RAG implementation.
2. Using audio sequencing AI models, I implemented a pipeline to process podcast audio and extract the voices of separate speakers. This audio was then used for training lip-sync animations using Style-Transfer and General Adverserial Networks.
